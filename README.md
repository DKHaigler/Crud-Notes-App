# Notes App (CRUD)

A simple notes application built using **React** and **Next.js** that demonstrates the core **CRUD** operations: create, read, update, and delete notes.

## Features
- Add and delete notes
- Edit existing notes
- Client-side state management
- Responsive UI structure

## Tech Stack
- Next.js
- React
- JavaScript (or TypeScript)
- CSS / SCSS
